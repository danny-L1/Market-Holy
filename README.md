# 프로젝트 기획의도

쇼핑몰은 회원관리나 상품목록 출력 등 수업시간에 배웠던 기능을 복습하고 확장하기 좋을 것 같아  세미 프로젝트 주제로 선정했고, 쇼핑몰 중 카테고리가 비교적 간단한 마켓컬리의 포맷을 따와 구현 했습니다.
